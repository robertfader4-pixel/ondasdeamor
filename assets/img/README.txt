Папка для ваших изображений.

Рекомендуемые имена файлов:
- cover.jpg  -> обложка на главной странице
- author.jpg -> фото автора
- chapter-01.jpg -> иллюстрация главы 1
- chapter-02.jpg -> иллюстрация главы 2
- chapter-03.jpg ... chapter-10.jpg -> иллюстрации остальных глав

Важно:
Сейчас сайт НЕ зависит от этих файлов и не ломается без них.
Когда захотите добавить изображение, замените соответствующий блок-плейсхолдер на тег <img>.
